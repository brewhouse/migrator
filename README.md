# Migrator

A web application to migrate content from an existing website page to a new development WordPress site.

## Features
- Enter source page URL(s)
- Enter destination WordPress URL and admin/editor credentials
- Option to migrate as page or post
- Option to migrate hero/banner image as featured image
- Images/PDFs from source page added to WordPress media library
- Extracts main content (excludes header, footer, menu, navigation)
- WordPress REST API integration (MCP if needed)
- Plugin recommendations for WordPress
- Styled like https://adachecker.planeteria.com/
- Deployable to Render
- GitHub integration
- Gravity Forms JSON export for forms
